# Workout Generator

Click the button, get a workout. Don't like it? Click it again.

![App Screenshot](workoutgeneratorscreenshot.png)

# Installation

```
git clone https://github.com/JedidiahCajudo/workout-generator.git
cd workout-generator
npm install
npm run dev
```
Finally, just open `http://localhost:5173`

Enjoy your workouts!
